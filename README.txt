?? Instalación

Clona el repositorio:


git clone https://github.com/tu-usuario/tattoo-catalog.git



Ingresa a la carpeta del proyecto:


cd tattoo-catalog

Instala las dependencias:


npm install

Inicia el servidor de desarrollo:


npm run dev
